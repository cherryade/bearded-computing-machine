Welcome to Wire

Wire is a take on a great app written in Javascript, Meteor. Wire is written in PHP and allows you to quickly start up your own site like Product Hunt, Reddit or Hacker News. Wire allows you to quickly set up your own community in minutes without any trouble installing any extra dependencies. Wire comes with a pre-built easy on the eyes flat designed theme which can be customised with ease. Wire also has MailChimp built in allowing you to easily keep your community updated on what's happening via email.

Wire was original built for, and currently used by, Codenews.io

Get Wire

$ cd /var/www/html
$ git clone https://github.com/cherryade/wire.git

Installation is easy, just head on over to yourwebsite.tld/install/. When you install you will be asked to input your database credentials, a few site settings, and you can also create your first admin user.
